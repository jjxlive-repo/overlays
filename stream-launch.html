Social Stream Ninja Dock
Aggregated chat + TTS relay. Pick a voice, then launch — the link is built fresh each time.
Two fixes vs. your original link, both confirmed against Social Stream Ninja's source (dock.html / tts.js): (1) swapped speech=en-GB for lang=en-GB — the speech param flips on a master "read every incoming message" fallback that would have spoken everything regardless of the . command or members-only setting; lang sets the same accent without that side effect. (2) dropped modtts — it restricts all TTS to moderator messages only (verified in source), which would have blocked members entirely rather than including mods. (3) added ttssources=youtube so SSN only ever speaks YouTube chat — TikTok already has its own TTS through Tikfinity, and this stops SSN from double-reading it.

Mods without a membership — dock.html's command gate checks channel membership only, never moderator status, and there's no URL parameter for it. Since comping isn't an option, use the Mod TTS Allowlist userscript below instead: it patches the dock's own message handler (client-side, in your browser) to stamp membership on names you add to a self-service list, so they pass the same gate a real paid member would.
